1.為什麼要學git

2.git的環境安裝跟環境設定指令
【環境安裝】
    1.安裝git、sourcetree
    2.github註冊

【環境安裝指令】
    git init :開新資料庫
    git --version :確認有沒有安裝好
    /*按小寫q跳出*/

【設定個人資料】
    1.輸入姓名：git config --global user.name "XXX"
    2.輸入個人的 email：git config --global user.email "XXX@gmail.com"
    3.查詢 git 設定內容：git config --list


3.git常用指令+要會畫基本指令關係圖

    git push -u origin master
